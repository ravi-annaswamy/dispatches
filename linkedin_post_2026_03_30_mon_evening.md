# LinkedIn/Twitter Highlights Post — Monday Evening Edition

---

Monday's X feed told a single story from seven angles: the developer's workstation is becoming an autonomous system, and every major player moved today.

Anthropic shipped computer use in Claude Code — the fourth iteration in 17 months, and as @aakashgupta noted, the first one that actually changes how developers work. Claude can now open your apps, click through your UI, and test what it built, all from the terminal. Meanwhile @claudeai expanded auto mode to Enterprise, letting the model make its own permission decisions. @lydiahallie then acknowledged what every power user already knew: people are hitting usage limits way faster than expected. 4.4K likes on that one. The demand is real.

OpenAI's response was strategically audacious. They launched a Codex plugin that runs *inside* Claude Code — embedding their cloud agent directly in Anthropic's developer tool. @ai_for_success captured the mood perfectly. And fresh data from @OpenAIDevs showed something fascinating: developers are using Codex as a night-shift engineer, delegating their hardest refactors and architecture work at the end of the workday. The shift from pair-programming to asynchronous delegation is quietly becoming the norm.

On the open-source front, @Hesamation surfaced a Qwen 3.5 27B model fine-tuned on Opus 4.6 distilled data that beats Sonnet 4.5 on SWE-bench — and runs locally on 16GB. It's been #1 trending on Hugging Face for three weeks. The pattern is now unmistakable: open-source models are drafting behind frontier proprietary research, and the gap is narrowing at speed.

@garrytan's weekly coding report might be the most revealing data point: 126 commits, 253K lines added, 115 AI coding sessions (113 Claude Code, 2 Gemini), 72-day shipping streak. The Y Combinator president is essentially running an engineering org where AI is the staff. @dr_cintas then surfaced the most complete Claude Code setup yet — 27 agents, 64 skills, 33 commands, all open source. The tooling layer around AI coding is maturing fast.

And in the background, ARC-AGI-3 humiliated every top model — none scored above 1%. A useful cold shower for a feed that otherwise reads like a capabilities victory lap.

The thread that connects all of this: we're watching the workstation evolve from a tool you operate into a system that operates on your behalf. The question isn't whether that's happening — Monday's feed made that clear. The question is who controls the operating layer.
